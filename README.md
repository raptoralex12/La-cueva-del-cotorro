La Cueva del Cotorro es un proyecto de realidad virtual de boxeo desarrollado como parte de una campaña de prevención de adicciones.Se busca combinar la emoción de los combates con un mensaje positivo para todo tipo de audiencia.

En el juego, el jugador enfrenta a El Cotorro, un duende que representa el uso de las drogas e intenta influenciar al jugador para consumir distintas sustancias.

Participé como programador principal, encargado del sistema de combate, los estados del enemigo y la integración general en Unity.

El proyecto fue presentado en eventos y ferias locales, incluyendo el Rodeo de Saltillo, como parte de una iniciativa gubernamental que promueve la prevención de adicciones mediante la tecnología y la realidad virtual.
